# Teenage Wrist

- **Origin:** Los Angeles, CA
- **Genre:** Alternative / shoegaze / grunge
- **Members:** Marshall Gallagher (vocals/guitar), Anthony Salazar (drums)
- **Fever LTD release:** _Dazed (2025 Remaster)_ — Vinyl (2 color variants: Neon Green/Black Swirl [sold out], Opaque Neon Green)
- **Notes:** Established act (also associated with Epitaph Records). Fuzzed-out, melodic alt sound.
